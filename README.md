# Gold-Grabber

Overview:
  This project tasks us with making a game using functions like abstarct classes, multi threading, and collision detection. We are meant to use ai to start the project, but due to my uncomfort with using ai I have decided to use pre-existing code as my reference points for the project. 

Project Function:
  The game I made is meant to start with a title screen that displays the name, the current high score, and the start button. When the stat button is pressed the screen would change to show a timer counting down with many coins bouncing around the screen. The player must click the coins to beat the game. After all coins are clicked the screen would change into an end screen with a message say how they did, and the core they got. If the score exceeded the current high score, it could placed in the text file.

Goals:
  My goal was to make a program that at least held the information that communicated the purpose of the code while using as many of the concepts I learned over the course as possible.
